# PhotonLink Setup Guide

## Prerequisites

| Tool | Version | Required |
|------|---------|----------|
| Flutter SDK | ≥ 3.22 (Dart ≥ 3.3) | Yes |
| Android Studio | Latest | Yes (Android first) |
| Android SDK | API 34, min API 21 | Yes |
| Rust toolchain | ≥ 1.70 | Optional (Phase 1 skeleton only) |
| Xcode | Latest | Optional (iOS future) |

## First-Time Setup

### 1. Install Flutter

Follow the official guide: https://docs.flutter.dev/get-started/install

Verify installation:

```bash
flutter doctor
```

### 2. Initialize Platform Files (if needed)

If platform folders are incomplete, run from the `photonlink_app/` directory:

```bash
cd photonlink_app
flutter create . --project-name photonlink_app --org com.photonlink
```

This merges missing platform scaffolding (icons, Gradle wrapper, etc.) without overwriting `lib/`.

### 3. Install Dependencies

```bash
cd photonlink_app
flutter pub get
```

### 4. Run on Android

Connect a device or start an emulator, then:

```bash
flutter run
```

### 5. Run Tests & Analysis

```bash
flutter analyze
flutter test
```

## Optional: Rust Core

The Rust crate at `native/photonlink_core/` is a standalone skeleton and is **not** wired to Flutter in Phase 1.

```bash
cd native/photonlink_core
cargo build
cargo test
```

```
├── native/
│   └── photonlink_core/  # Rust core skeleton
└── docs/
```

## Troubleshooting

### Camera permission denied
Grant camera permission in device Settings, or tap "Open Settings" on the scan screen.

### Gradle build fails
Ensure `local.properties` contains your Android SDK path (auto-generated by Flutter).

### Missing platform icons
Run `flutter create .` as described in step 2.

## Current Limitations

- Rust core (`native/photonlink_core/`) is not connected via FFI
- LZ4 compression is a disabled placeholder (GZip is active)
- Optical Stream and Audio transports are UI stubs only
- Color Matrix uses one-way cyclic broadcast (no ACK/NAK feedback channel)

See [AUDIT_PHASE1-5.md](AUDIT_PHASE1-5.md) for the full Phase 1–5 audit.
